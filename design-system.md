# design-system.md — 資料化システム 共通デザイントークン

このファイルは `document-system` 配下の全てのskill（AGENT.md本体、`skills/*.md`）から参照される、**唯一のデザイン基準（Single Source of Truth）**である。矛盾する記述が他にあっても、必ずこちらを優先する。

## 大原則

**色は土台色＋白＋グレーのみ。アクセントカラーは使わない。**
強調は「色を増やす」のではなく、**濃淡（ネイビーの塗りつぶし vs グレーの塗りつぶし）と太字・サイズ**で行う。図やグラフが増えても、資料全体としての色数は増えない。

## カラートークン

```css
:root {
  --navy:       #1B2436;  /* 土台・信頼。表紙/最終スライド背景、タイトル文字、強調ボックスの塗り */
  --white:      #FFFFFF;  /* 本文スライド背景 */
  --gray-lt:    #F2F3F5;  /* カード・ボックスの通常時の塗り */
  --gray-border:#D8DAE0;  /* ボックス枠線、罫線 */
  --gray-mid:   #9AA3B5;  /* 補足・キャプション・矢印・非強調要素 */
  --text:       #222222;  /* 本文 */
  --text-sub:   #555555;  /* 補足説明文 */
  --white-on-navy: #FFFFFF; /* ネイビー背景上の文字 */
}
```

**配色比率の目安**：白ベース70% / ネイビー・グレー28% / それ以外 使わない（0%）

### 強調のルール（色を増やさずに強調する方法）
- **状態の強調**：注目させたい1要素だけ `--navy` で塗りつぶし、文字を白にする。他は `--gray-lt` 塗り
- **文字の強調**：太字（bold）とサイズ差で優先度を表現する
- **警告・課題の強調**：色を変えるのではなく、ボックスの枠線を太くする（例：`border: 2px solid var(--navy)`）か、「要判断」「課題」という文字ラベルを添える
- グラデーション・影・装飾バー・複数色の使用は禁止（フラット厳守）

## タイポグラフィ

```css
:root {
  --font: "Meiryo", "MS PGothic", sans-serif;  /* メイリオ固定。フォールバックのみ用意 */
  --title-size:   28px;  /* スライドタイトル Bold */
  --heading-size: 18px;  /* セクション見出し Bold */
  --body-size:    14px;  /* 本文 */
  --caption-size: 11px;  /* 補足 グレー */
}
```

- 日本語フォントは **メイリオに固定**。他フォントへの変更は不可（BIZ UDゴシック等への差し替えは検討済みの上で不採用）
- pptx変換時のフォント指定も `"Meiryo"` で統一する（`font.name = "Meiryo"`）

## 余白・レイアウト

```css
:root {
  --margin:  64px;   /* スライド端マージン */
  --gap:     40px;   /* ブロック間 */
  --pad:     16px;   /* ボックス内パディング */
  --radius:  4px;    /* 角丸は控えめに（consult-deckの8pxより少し硬めの4px） */
}
```

- アスペクト比：16:9（1280×720px基準、pptx変換時は13.333×7.5インチ）
- 上部にタイトルエリア（高さ約72px）、下にコンテンツエリアの2段構成
- タイトルは28px Bold・ネイビー・左揃え。**結論文（メッセージライン）にする**（例：「原因分析」ではなく「原因は◯◯の可能性が高い」）
- 表紙・最終スライドのみネイビー背景＋白文字。中間スライドは白背景
- フッターに日付・ページ番号（11px・グレー・右下）

## HTML→pptx 変換の共通対応表

| HTML要素 | pptx実装 |
|---------|---------|
| `<section class="slide">` | 1スライド（16:9 = 13.333×7.5インチ） |
| `--navy` `--white` `--gray-lt` 等 | `RGBColor(0x1B,0x24,0x36)` のように直接変換 |
| タイトル28px | `Pt(20)` 前後 |
| 本文14px | `Pt(11)` 前後 |
| `--margin: 64px` | 左右上下 `Inches(0.5)` |
| ボックス（`.box` 系） | 角丸矩形 `MSO_SHAPE.ROUNDED_RECTANGLE` |
| ネイビー背景スライド | スライド背景fill = navy、文字 = white |
| フォント指定 | 全テキストボックスで `font.name = "Meiryo"` を明示指定 |

> px→インチはHTMLが1280px=13.333インチ換算（96dpi）。

## 各skillでの参照方法

各 `skills/*.md` の冒頭で「本skillのカラー・フォント・余白は `design-system.md` に従う」と明記し、個別に色やフォントを再定義しないこと。もし個別skillの実装例にコード片で色を書く場合も、必ずこのファイルの値と一致させる。
